# Plano 05: Mover e Reordenar Post-its

## Objetivo
Implementar funcionalidade de mover post-its entre posições do grid

## Tarefas

### 5.1 Modo de Movimentação
- [ ] Binding `m` - Entrar em modo "Move"
- [ ] Indicador visual do modo move ativo
- [ ] Selecionar post-it para mover
- [ ] Setas direcionais para mover post-it

### 5.2 Lógica de Troca
- [ ] Implementar swap entre duas posições
- [ ] Mover post-it para posição vazia
- [ ] Trocar post-its de posição
- [ ] Atualizar propriedade `position` de ambos

### 5.3 Feedback Visual
- [ ] Destacar post-it sendo movido
- [ ] Indicar posição de destino
- [ ] Animação ou efeito de movimento
- [ ] Confirmar com Enter, cancelar com Escape

### 5.4 Alternativa: Drag & Drop
- [ ] (Opcional) Se Textual suportar, implementar drag & drop
- [ ] Clicar e arrastar post-it
- [ ] Soltar em nova posição

### 5.5 Atalhos de Posição
- [ ] Números 1-9: mover post-it selecionado para posição específica
- [ ] Shift + números: trocar posições diretamente

## Resultado Esperado
Usuário pode reorganizar post-its livremente entre as 9 posições do grid
